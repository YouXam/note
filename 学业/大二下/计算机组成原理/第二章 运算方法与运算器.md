
# 已知 $[X]_{补}$ 求 $[-X]_补$

连同符号位一起，各位按位取反，末位加1。
# 机器码右移位

1. 原码：直接右移
2. 原码：符号为固定，左边空出的数值补 0
3. 补码和反码：符号位固定，左边空出的数值补符号位

# 补码位数扩展

- 定点纯整数：补码符号位固定在最高位，左边空出的数值补符号位。
	**1**010 -> **1111**010
- 定点纯小数：补码右边空出的数值补 0
	1.100 -> 1.100**0000**

# 移码表示法

浮点数中，**阶**常使用移码表示法。

$$[x]_{\text{移}}=2^n+x, \text{ 其中 }n\text{ 满足 }2^n > x \ge -2^n$$

移码和补码的符号位相反，其他位相同。
